# BadClockPlus

This is a derrivative of [BadClock](https://github.com/NielsLeenheer/BadClock/), an awesome hardware/software project by Niels Leenheer.  [See it in action here](https://front-end.social/@html5test/116318323518245083), and check the repo for the core build instructions.

This project adds some extra silliness to the original BadClock.  Every five minutes [something funny happens](https://flyingtoasters.org).

It incorporates [another cursed clock](https://boat.horse/clock/index.html) via Ryan Bateman.
